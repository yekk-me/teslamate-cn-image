## 🚀 快速启动 Quick Start

### 部署步骤 Deployment

```bash
# 克隆仓库
git clone https://github.com/gococonut/teslamate-cn-docker.git
cd teslamate-cn-docker

# 配置环境变量
cp .env.example .env
nano .env  # 修改环境变量

# 启动服务
docker-compose up -d
```

### 云服务器一键部署

#### 准备工作

1. 购买腾讯云服务器
推荐购买腾讯云新加坡轻量应用服务器：  
配置: 1核2GB，99元/年  
地区: 新加坡（海外节点，访问 Tesla API 更稳定）
购买链接: 腾讯云活动
服务器规格建议: 
内存: 2GB 以上  
存储: 40GB 以上 
系统：Ubuntu 24.04 LTS
2. 购买域名
在腾讯云购买域名： 
    - 进入腾讯云控制台 
3. 选择"域名注册" 
    - 搜索并购买一个便宜的域名（如 .top、.xyz 等后缀）
    - 完成实名认证
4. 配置 DNS 解析
    - 添加 A 记录
    - 在腾讯云 DNS 解析控制台：
        - 主机记录: teslamate（或您喜欢的子域名）
        - 记录类型: A
        - 记录值: 您的服务器公网 IP
        - TTL: 600
